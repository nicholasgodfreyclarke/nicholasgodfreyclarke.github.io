Graphbattle tutorial

Graphbattle is a turn based game in which you must write functions to hit your enemies. The functions will start from the active soldier (always located at the origin). Navigate through the obstacles and avoid hitting your own team!

Available functions

  • sin
  • cos
  • tan
  • exp
  • sqrt
  • abs
  • floor
  • ceil
  • max
  • min
  • ln
  • log
  • asin
  • acos
  • atan
  • sec
  • csc
  • cot
  • asec
  • acsc
  • acot
  • sinh
  • cosh
  • tanh
  • asinh
  • acosh
  • atanh
  • sech
  • csch
  • coth
  • asech
  • acsch
  • acoth

Examples

  • 2x
  • sin(10x)/2 + x^2
  • ln(abs(x-5))
  • (1/x)^sqrt(1.25x)

Tips

  • To allow the function to start from the firing player - the function will be translated up or down to the player's y coordinate. A consequence being that the addition of constants to the function will have no effect.
  • Each "soldier" stores a history of functions it has used in the game - you can cycle through them by pressing up or down on the keyboard. Useful for fine tuning shots.
  • Use the axes to help calculate the right function - they are reset each turn.
